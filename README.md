# Charts

## Usage

### Releases

```
helm repo add linefusion https://linefusion.github.io/charts
helm repo update
```

### Libraries

```
dependencies:
- name: <chart-name>
  version: "<chart-version>"
  repository: "https://linefusion.github.io/charts"
```
